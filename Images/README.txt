INSTRUÇÕES PARA O LOGO
======================

1. Coloque o arquivo de imagem do logo nesta pasta com o nome:
   logo-kingdom-confeitaria.png

2. Formatos aceitos: PNG, JPG, SVG
   - Recomendado: PNG com fundo transparente ou verde escuro (#1a4d2e)
   - Tamanho recomendado: largura mínima de 400px
   - Proporção: mantida automaticamente

3. Se o arquivo tiver outro nome, você pode:
   - Renomear para: logo-kingdom-confeitaria.png
   - OU editar o arquivo Default.aspx e alterar o caminho na linha:
     <img src="Images/logo-kingdom-confeitaria.png" ... />

4. Se a imagem não for encontrada, o sistema mostrará automaticamente
   um texto alternativo com o nome "Kingdom Confeitaria".
