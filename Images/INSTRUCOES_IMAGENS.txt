INSTRUÇÕES PARA ADICIONAR IMAGENS DOS PRODUTOS

SACOS PROMOCIONAIS:
------------------

- saco-3-grandes.jpg

IMAGEM PADRÃO (FALLBACK):
------------------------
- placeholder.png (será usada se a imagem do produto não for encontrada)

FORMATO RECOMENDADO:
-------------------
- Formato: JPG ou PNG
- Tamanho recomendado: 400x400 pixels ou maior
- Peso: máximo 500KB por imagem
- Qualidade: boa qualidade, mas otimizada para web

COMO ADICIONAR:
--------------
1. Prepare as imagens dos produtos
2. Renomeie cada imagem conforme a lista acima
3. Coloque todas as imagens nesta pasta (Images/)
4. Se uma imagem não for encontrada, será usado o placeholder.png

NOTA:
-----
Se você não tiver todas as imagens, o sistema usará o placeholder.png automaticamente.
Você pode adicionar as imagens gradualmente conforme disponível.
